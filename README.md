\# 🎓 Student Management System (Spring Core)



A simple Student Management System built using Spring Core to understand Dependency Injection and Layered Architecture.



\---



\## 🚀 Technologies Used



\- Java 17

\- Spring Framework 6

\- Maven

\- Eclipse IDE



\---



\## 📚 Concepts Covered



\- IoC Container

\- Dependency Injection

\- Constructor Injection

\- @Component

\- @Service

\- @Repository

\- @Controller

\- @Autowired

\- @Primary

\- @Qualifier

\- Bean Scope (Singleton \& Prototype)

\- Bean Lifecycle

\- Layered Architecture



\---



\## 📂 Project Structure



```

Controller

&#x20;    ↓

Service

&#x20;    ↓

Repository

&#x20;    ↓

Model

```
